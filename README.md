# Detection of Vitamin Deficiency and Disease through Facial Analysis

## Overview

This project uses Artificial Intelligence and Machine Learning to detect vitamin deficiencies and diseases through facial analysis. It analyzes facial features such as skin tone, lips, and eyes to identify possible health conditions.

## Key Features

* Facial image analysis
* Vitamin deficiency detection
* Disease prediction
* AI-based classification
* Fast and non-invasive detection
* User-friendly system

## Tools and Technology

* MATLAB
* Machine Learning
* Support Vector Machine (SVM)
* Histogram of Oriented Gradients (HOG)
* Image Processing
* Computer Vision

## Design Flow

* Collect facial images
* Extract facial features
* Train the ML model
* Test the input image
* Detect vitamin deficiency
* Display the result
